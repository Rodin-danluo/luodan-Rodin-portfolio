# 罗丹个人作品集网站

## 🎯 项目概述

这是一个现代化的个人作品集网站，专为UI/UX设计师求职而设计。网站以橙色为主题色，具有强烈的交互性和视觉冲击力。

## ✨ 核心特性

### 🎨 视觉设计
- **主题色彩**：橙色系 (#FF6B35, #F7931E, #FF8C42)
- **毛玻璃效果**：首屏整体模糊，悬浮名字区域时变清晰
- **响应式设计**：完美适配桌面端、平板和移动端
- **现代动效**：流畅的过渡动画和微交互

### 🚀 交互功能
- **自定义光标**：跟随鼠标的动态光标效果
- **滚动动画**：元素进入视窗时的渐入动画
- **数据可视化**：技能雷达图和成果数据展示
- **项目详情页**：点击项目卡片跳转到详细展示页面

### 📱 页面结构
```
首页 (index.html)
├── 英雄区域 - 毛玻璃名字效果
├── 关于我 - 教育背景和个人特质
├── 项目展示 - UI/UX设计作品
├── 综合能力 - 运营数据和其他技能
├── 技能展示 - 可视化技能图表
└── 联系方式 - 社交媒体链接

项目详情页 (project-detail.html)
├── 项目概览
├── 设计过程
├── 项目成果
├── 学习收获
└── 项目导航
```

## 🖼️ 图片使用指南

### 个人照片 (images/profile/)
请将你的个人照片按以下命名规则放入 `images/profile/` 文件夹：

- `profile-main.jpg` - 主要头像（用于首屏和导航）
- `profile-about.jpg` - 关于我页面的照片
- `profile-casual.jpg` - 生活照片（展现个性）

**建议规格**：
- 主头像：400x400px，正方形
- 关于我照片：600x400px，横向
- 生活照：自然比例即可

### 项目作品图片 (images/projects/)
按项目类型分类存放：

```
images/projects/
├── ui-ux/
│   ├── social-app-main.jpg     # 社交APP主图
│   ├── social-app-1.jpg        # 详情页图片1
│   ├── social-app-2.jpg        # 详情页图片2
│   └── ...
├── mobile/
│   ├── ecommerce-mobile-main.jpg
│   └── ...
├── web/
│   ├── admin-dashboard-main.jpg
│   └── ...
└── prototype/
    ├── health-app-main.jpg
    └── ...
```

**建议规格**：
- 主图：1200x800px
- 详情图：800x600px 或实际截图尺寸

### 教育背景图片 (images/education/)
- `university-logo.png` - 湖南工业大学校徽
- `lab-photo.jpg` - IDL创新实验室照片
- `activities.jpg` - 班级活动或领导工作照片

### 获奖证书 (images/awards/)
- `award-1.jpg`, `award-2.jpg` 等 - 各类获奖证书
- 建议扫描为高清图片，保持证书的清晰度

- **CSS3** - Flexbox、Grid、动画和响应式设计

## 🚀 部署说明

### 本地预览
1. 将所有文件放在同一目录下
2. 按照图片指南放置图片文件
3. 使用本地服务器打开 `index.html`

### 推荐部署平台
- **Netlify** - 拖拽部署，自动HTTPS
- **Vercel** - 现代化部署平台
- **GitHub Pages** - 免费静态网站托管

## 📝 内容定制

### 个人信息修改
在 `index.html` 中修改以下内容：
- 个人姓名和英文名
- 教育背景信息
- 联系方式和社交媒体链接
- 技能列表和熟练度

### 项目数据更新
在 `js/project-detail.js` 中的 `projectsData` 对象中：
- 添加新项目数据
- 修改现有项目信息
- 更新项目图片路径
- 调整项目导航顺序

### 样式定制
在 `css/style.css` 中可以修改：
- 主题色彩变量
- 字体选择
- 动画效果
- 响应式断点

## 🎨 设计理念

### 色彩心理学
- **橙色主题**：代表活力、创新、温暖
- **体现性格**：开朗、外向、有活力
- **专业感**：深色文字保持专业性

### 交互设计
- **毛玻璃效果**：营造神秘感和现代感
- **悬浮交互**：增强用户参与感
- **流畅动画**：提升用户体验

### 信息架构
- **用户为中心**：突出求职目标和能力
- **故事化叙述**：从个人介绍到项目展示的完整故事线
- **数据驱动**：用具体数据证明能力和成果

## 📞 技术支持

如果在使用过程中遇到问题，可以：
1. 检查浏览器控制台的错误信息
2. 确认图片路径是否正确
3. 验证HTML和CSS语法
4. 测试不同浏览器的兼容性
